Developed an interactive browser-based mini game hub using HTML, CSS, and vanilla JavaScript. The platform allows users to play multiple games within a single interface, featuring modular design, responsive layout, and enhanced user experience with animations and sound effects.

Tech Stack

  • HTML5
  • CSS3
  • JavaScript (Vanilla JS)

Key Features

  • Multiple mini-games in a single platform
  • Modular JavaScript architecture
  • Responsive design for all devices
  • Interactive UI with animations
  • Sound effects for better user engagement

How It Works
