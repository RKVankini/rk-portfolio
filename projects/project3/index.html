CI/CD Pipeline with Jenkins, Terraform, Docker & ArgoCD

Built a complete end-to-end DevOps pipeline to automate application deployment using Jenkins for CI, Docker for containerization, Terraform for infrastructure provisioning, and ArgoCD for GitOps-based deployment to Amazon EKS. The pipeline integrates security scanning and continuous delivery practices.

Tech Stack

  • AWS (EKS, EC2, ECR)
  • Jenkins (CI/CD Automation)
  • Docker (Containerization)
  • Terraform (Infrastructure as Code)
  • ArgoCD (GitOps Deployment)
  • SonarQube (Code Quality Analysis)
  • Trivy (Security Scanning)

CI/CD Workflow

  • Developer pushes code to GitHub
  • Jenkins triggers pipeline and builds application
  • SonarQube analyzes code quality
  • Docker builds and tags container image
  • Trivy scans image for vulnerabilities
  • Terraform provisions infrastructure (EKS cluster)
  • ArgoCD deploys application to Kubernetes (GitOps)

Key Features

  • Fully automated CI/CD pipeline
  • Infrastructure provisioning using Terraform
  • GitOps-based deployment with ArgoCD
  • Integrated security scanning using Trivy
  • Code quality checks using SonarQube
  • Scalable deployment on AWS EKS

Key Achievements

  • Reduced manual deployment effort significantly
  • Improved deployment consistency and reliability
  • Implemented DevSecOps practices
  • Automated infrastructure and application lifecycle
