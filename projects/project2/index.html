RK Weather Dashboard

A lightweight and responsive weather application built using vanilla JavaScript. The app fetches real-time weather data from an external API and presents it with a clean, user-friendly interface focused on performance and usability.

🚀 View Live Demo

Tech Stack

  • HTML5
  • CSS3
  • JavaScript (Vanilla JS)
  • OpenWeather API

Key Features

  • Real-time weather data fetching using API
  • Search functionality for different cities
  • Responsive UI for mobile and desktop
  • Fast loading and minimal design
  • Error handling for invalid inputs

How It Works

  • User enters a city name
  • Application sends request to weather API
  • Receives JSON response
  • Displays temperature, conditions, and location details

Key Achievements

  • Built a clean and responsive frontend application
  • Integrated external API successfully
  • Focused on performance and user experience
