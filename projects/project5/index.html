Hotstar DevOps Project (Node.js)

Implemented a complete DevOps pipeline to deploy a Node.js-based OTT platform (Hotstar clone) using Jenkins, Docker, and Kubernetes. The project demonstrates real-world CI/CD practices, containerization, and scalable application deployment on cloud infrastructure.

Tech Stack

  • Node.js (Application)
  • Jenkins (CI/CD Automation)
  • Docker (Containerization)
  • Kubernetes (Deployment)
  • AWS (EKS / EC2)
  • Git & GitHub

CI/CD Workflow

  • Code pushed to GitHub repository
  • Jenkins triggers build pipeline
  • Docker image is created from Node.js app
  • Image pushed to container registry
  • Kubernetes deploys application to cluster

Key Features

  • End-to-end CI/CD pipeline implementation
  • Containerized Node.js application
  • Scalable deployment using Kubernetes
  • Automated build and deployment process
